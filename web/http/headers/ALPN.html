    ALPN

    Used by user agents to specify response media types that are acceptable.

    Overview table

    Name
    ALPN
    Direction
    Request
    Specification
    RFC 7639: The ALPN HTTP Header Field

    Features:

    Syntax

    ALPN            = 1#protocol-id
protocol-id     = token ; percent-encoded ALPN protocol identifier

    Examples

    
CONNECT www.example.com HTTP/1.1
Host: www.example.com
ALPN: h2, http%2F1.1
